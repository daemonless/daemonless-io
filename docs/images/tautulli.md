## Deployment


=== ":material-docker: Podman Compose"

    ```yaml
    services:
      tautulli:
        image: @REGISTRY@/tautulli:latest
        container_name: tautulli
        environment:
          - PUID=@PUID@
          - PGID=@PGID@
          - TZ=@TZ@
          - TAUTULLI_DOCKER=True
        volumes:
          - "@CONTAINER_CONFIG_ROOT@/@TAUTULLI_CONFIG_PATH@:/config"
        ports:
          - @TAUTULLI_PORT@:8181
        restart: unless-stopped
    ```
