ReadMeABook bundles PostgreSQL, which requires `allow.sysvipc` for shared memory. Create `readmeabook.conf` alongside `appjail-director.yml`:

```
exec.start: "/bin/sh /etc/rc"
exec.stop: "/bin/sh /etc/rc.shutdown jail"
mount.devfs
persist
allow.sysvipc
```
