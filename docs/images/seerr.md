## Deployment


=== ":material-docker: Podman Compose"

    ```yaml
    services:
      seerr:
        image: @REGISTRY@/seerr:latest
        container_name: seerr
        environment:
          - PUID=@PUID@
          - PGID=@PGID@
          - TZ=@TZ@
        volumes:
          - "@CONTAINER_CONFIG_ROOT@/@SEERR_CONFIG_PATH@:/config"
        ports:
          - @SEERR_PORT@:5055
        restart: unless-stopped
    ```

