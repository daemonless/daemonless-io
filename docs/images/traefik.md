## Deployment


=== ":material-docker: Podman Compose"

    ```yaml
    services:
      traefik:
        image: @REGISTRY@/traefik:latest
        container_name: traefik
        environment:
          - PUID=@PUID@
          - PGID=@PGID@
          - TZ=@TZ@
        volumes:
          - "@CONTAINER_CONFIG_ROOT@/@TRAEFIK_CONFIG_PATH@:/config"
        ports:
          - @TRAEFIK_PORT@:80
          - 443:443
          - 8080:8080
        restart: unless-stopped
    ```


=== ":appjail-appjail: AppJail Director"

    **.env**:

    ```
    DIRECTOR_PROJECT=traefik
    PUID=@PUID@
    PGID=@PGID@
    TZ=@TZ@
    ```

    **appjail-director.yml**:

    ```yaml
    options:
      - virtualnet: ':<random> default'
      - nat:
    services:
      traefik:
        name: traefik
        options:
          - container: 'boot args:--pull'
        oci:
          user: root
          environment:
            - PUID: !ENV '${PUID}'
            - PGID: !ENV '${PGID}'
            - TZ: !ENV '${TZ}'
        volumes:
          - TRAEFIK_CONFIG_PATH: /config
    volumes:
      TRAEFIK_CONFIG_PATH:
        device: '@CONTAINER_CONFIG_ROOT@/@TRAEFIK_CONFIG_PATH@'
    ```

    **Makejail**:

    ```
    ARG tag=latest

    OPTION overwrite=force
    OPTION from=@REGISTRY@/traefik:${tag}
    ```

## Parameters

### Environment Variables

| Variable | Default | Description |
|----------|---------|-------------|
| `PUID` | `1000` | User ID for the application process |
| `PGID` | `1000` | Group ID for the application process |
| `TZ` | `UTC` | Timezone for the container |

### Volumes

| Path | Description |
|------|-------------|
| `/config` | Configuration directory (traefik.yml, dynamic/, letsencrypt/) |

### Ports

| Port | Protocol | Description |
|------|----------|-------------|
| `80` | TCP | HTTP |
| `443` | TCP | HTTPS |
| `8080` | TCP | Dashboard/API |


!!! info "Implementation Details"

    - **Architectures:** amd64
    - **User:** `bsd` (UID/GID set via [PUID/PGID](../guides/permissions.md)). Defaults to `1000:1000`.
    - **Base:** Built on `@REGISTRY@/base` (FreeBSD 15.0).
