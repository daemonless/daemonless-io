## Deployment


=== ":material-docker: Podman Compose"

    ```yaml
    services:
      plex:
        image: @REGISTRY@/plex:latest
        container_name: plex
        environment:
          - PUID=@PUID@
          - PGID=@PGID@
          - TZ=@TZ@
          - VERSION=container
          - PLEX_CLAIM=
        volumes:
          - "@CONTAINER_CONFIG_ROOT@/@PLEX_CONFIG_PATH@:/config"
          - "@CONTAINER_CONFIG_ROOT@/@PLEX_TRANSCODE_PATH@:/transcode" # optional
          - "/path/to/@MOVIES_PATH@:/movies"
          - "/path/to/@TV_PATH@:/tv"
        ports:
          - @PLEX_PORT@:32400
          - 1900:1900
          - 32410:32410
          - 32412:32412
          - 32413:32413
          - 32414:32414
          - 32469:32469
          - 8324:8324
        restart: unless-stopped
    ```

## Parameters

### Environment Variables

| Variable | Default | Description |
|----------|---------|-------------|
| `PUID` | `1000` | User ID for the application process |
| `PGID` | `1000` | Group ID for the application process |
| `TZ` | `UTC` | Timezone for the container |
| `VERSION` | `container` | Plex update channel (container, public, plexpass) |
| `PLEX_CLAIM` | `` | Optional: Claim token from https://plex.tv/claim |

### Volumes

| Path | Description |
|------|-------------|
| `/config` | Configuration directory |
| `/transcode` | Transcode directory (Optional) |
| `/movies` | Movie library |
| `/tv` | TV series library |
