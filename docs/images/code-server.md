## Deployment


=== ":material-docker: Podman Compose"

    ```yaml
    services:
      code-server:
        image: @REGISTRY@/code-server:latest
        container_name: code-server
        environment:
          - PUID=@PUID@
          - PGID=@PGID@
          - TZ=@TZ@
        volumes:
          - "@CONTAINER_CONFIG_ROOT@/@CODE_SERVER_CONFIG_PATH@:/config"
        ports:
          - @CODE_SERVER_PORT@:8080
        restart: unless-stopped
    ```

> **Work in Progress**: This image is functional but may change significantly.

Common dev tools (gcc, clang, llvm, python, gmake) are baked into the image for now.
We are evaluating better approaches for user-managed package installation that survive image upgrades —
the current toolset and how it is delivered may change in a future release.

The `bsd` user has passwordless `doas` access, so you can install additional packages from the terminal:

```sh
doas pkg install rust cargo
```



!!! info "Implementation Details"

    - **Architectures:** amd64
    - **User:** `bsd` (UID/GID set via [PUID/PGID](../guides/permissions.md)). Defaults to `1000:1000`.
    - **Base:** Built on `@REGISTRY@/base` (FreeBSD 15.0).
