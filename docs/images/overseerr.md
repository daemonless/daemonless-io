## Deployment


=== ":material-docker: Podman Compose"

    ```yaml
    services:
      overseerr:
        image: @REGISTRY@/overseerr:latest
        container_name: overseerr
        environment:
          - PUID=@PUID@
          - PGID=@PGID@
          - TZ=@TZ@
        volumes:
          - "@CONTAINER_CONFIG_ROOT@/@OVERSEERR_CONFIG_PATH@:/config"
        ports:
          - @OVERSEERR_PORT@:5055
        restart: unless-stopped
    ```
