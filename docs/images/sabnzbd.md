## Deployment


=== ":material-docker: Podman Compose"

    ```yaml
    services:
      sabnzbd:
        image: @REGISTRY@/sabnzbd:latest
        container_name: sabnzbd
        environment:
          - PUID=@PUID@
          - PGID=@PGID@
          - TZ=@TZ@
        volumes:
          - "@CONTAINER_CONFIG_ROOT@/@SABNZBD_CONFIG_PATH@:/config"
          - "/path/to/@DOWNLOADS_PATH@:/downloads"
        ports:
          - @SABNZBD_PORT@:8080
        restart: unless-stopped
    ```


=== ":material-console: Podman CLI"

    ```bash
    podman run -d --name sabnzbd \
      -p @SABNZBD_PORT@:8080 \
      -e PUID=@PUID@ \
      -e PGID=@PGID@ \
      -e TZ=@TZ@ \
      -v @CONTAINER_CONFIG_ROOT@/@SABNZBD_CONFIG_PATH@:/config \
      -v /path/to/@DOWNLOADS_PATH@:/downloads \
      @REGISTRY@/sabnzbd:latest
    ```

=== ":simple-ansible: Ansible"

    ```yaml
    - name: Deploy sabnzbd
    ```
