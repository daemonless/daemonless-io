!!! failure "Root Privileges Required"
    **Podman on FreeBSD currently requires root.** All commands must be run as root (or via `doas`/`sudo`).

## Deployment


=== ":material-docker: Podman Compose"

    ```yaml
    services:
      unifi:
        image: @REGISTRY@/unifi:latest
        container_name: unifi
        environment:
          - PUID=@PUID@
          - PGID=@PGID@
          - TZ=@TZ@
        volumes:
          - "@CONTAINER_CONFIG_ROOT@/@UNIFI_CONFIG_PATH@:/config"
        ports:
          - @UNIFI_PORT@:8443
          - 8080:8080
          - 8843:8843
          - 8880:8880
          - 6789:6789
          - 3478:3478
          - 10001:10001
        annotations:
          org.freebsd.jail.allow.mlock: "true"
        restart: unless-stopped
    ```


=== ":appjail-appjail: AppJail Director"

    **.env**:

    ```
    DIRECTOR_PROJECT=unifi
    PUID=@PUID@
    PGID=@PGID@
    TZ=@TZ@
    ```

    **appjail-director.yml**:

    ```yaml
    options:
      - virtualnet: ':<random> default'
      - nat:
    services:
      unifi:
        name: unifi
        options:
          - container: 'boot args:--pull'
        oci:
          user: root
          environment:
            - PUID: !ENV '${PUID}'
            - PGID: !ENV '${PGID}'
            - TZ: !ENV '${TZ}'
        volumes:
          - UNIFI_CONFIG_PATH: /config
    volumes:
      UNIFI_CONFIG_PATH:
        device: '@CONTAINER_CONFIG_ROOT@/@UNIFI_CONFIG_PATH@'
    ```

    **Makejail**:

    ```
    ARG tag=latest

    OPTION overwrite=force
    OPTION from=@REGISTRY@/unifi:${tag}
    SET allow.mlock=1
    ```
