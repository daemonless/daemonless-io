## Deployment


=== ":material-docker: Podman Compose"

    ```yaml
    services:
      openspeedtest:
        image: @REGISTRY@/openspeedtest:latest
        container_name: openspeedtest
        environment:
          - PUID=@PUID@
          - PGID=@PGID@
          - TZ=@TZ@
        ports:
          - @OPENSPEEDTEST_PORT@:3000
        restart: unless-stopped
    ```


=== ":appjail-appjail: AppJail Director"

    **.env**:

    ```
    DIRECTOR_PROJECT=openspeedtest
    PUID=@PUID@
    PGID=@PGID@
    TZ=@TZ@
    ```

    **appjail-director.yml**:

    ```yaml
    options:
      - virtualnet: ':<random> default'
      - nat:
    services:
      openspeedtest:
        name: openspeedtest
        options:
          - container: 'boot args:--pull'
        oci:
          user: root
          environment:
            - PUID: !ENV '${PUID}'
            - PGID: !ENV '${PGID}'
            - TZ: !ENV '${TZ}'
    ```

    **Makejail**:

