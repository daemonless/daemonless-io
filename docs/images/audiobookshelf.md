## Deployment


=== ":material-docker: Podman Compose"

    ```yaml
    services:
      audiobookshelf:
        image: @REGISTRY@/audiobookshelf:latest
        container_name: audiobookshelf
        environment:
          - PUID=@PUID@
          - PGID=@PGID@
          - TZ=@TZ@
        volumes:
          - "@CONTAINER_CONFIG_ROOT@/@AUDIOBOOKSHELF_CONFIG_PATH@:/config"
          - "@CONTAINER_CONFIG_ROOT@/@AUDIOBOOKSHELF_METADATA_PATH@:/metadata"
          - "@CONTAINER_CONFIG_ROOT@/@AUDIOBOOKSHELF_AUDIOBOOKS_PATH@:/audiobooks"
        ports:
          - @AUDIOBOOKSHELF_PORT@:13378
        restart: unless-stopped
    ```

## Parameters

### Environment Variables

| Variable | Default | Description |
|----------|---------|-------------|
| `PUID` | `1000` | User ID for the application process |
| `PGID` | `1000` | Group ID for the application process |
| `TZ` | `UTC` | Timezone for the container |

### Volumes

| Path | Description |
|------|-------------|
| `/config` | Configuration and database |
| `/metadata` | Metadata cache |
| `/audiobooks` | Audiobook library path |

| Port | Protocol | Description |

!!! info "Implementation Details"

    - **Architectures:** amd64
    - **User:** `bsd` (UID/GID set via [PUID/PGID](../guides/permissions.md)). Defaults to `1000:1000`.
    - **Base:** Built on `@REGISTRY@/base` (FreeBSD 15.0).
