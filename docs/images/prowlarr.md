## Deployment


=== ":material-docker: Podman Compose"

    ```yaml
    services:
      prowlarr:
        image: @REGISTRY@/prowlarr:latest
        container_name: prowlarr
        environment:
          - PUID=@PUID@
          - PGID=@PGID@
          - TZ=@TZ@
        volumes:
          - "@CONTAINER_CONFIG_ROOT@/@PROWLARR_CONFIG_PATH@:/config"
        ports:
          - @PROWLARR_PORT@:9696
        annotations:
          org.freebsd.jail.allow.mlock: "true"
        restart: unless-stopped
    ```
