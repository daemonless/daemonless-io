## Contributing

We welcome contributions of all kinds:

- **Bug reports** - Found an issue? Open a GitHub issue
- **Feature requests** - Have an idea? Start a discussion
- **New images** - Want to add a container? See our [Development Guide](guides/development.md)
- **Documentation** - Help improve these docs

## Support

For help running daemonless containers:

1. Check the [Getting Started](quick-start.md) guide
2. Read the relevant [Guides](guides/permissions.md)
3. Search existing [GitHub Issues](https://github.com/daemonless/daemonless/issues)
4. Ask on [Discord](https://discord.gg/PTg5DJ2y)
